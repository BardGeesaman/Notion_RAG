# Navigate to OAuth Consent Screen

You're on a dashboard page. Here's how to get to the OAuth consent screen configuration:

## Method 1: Left Sidebar Navigation

1. Look at the **left side** of the page
2. You should see a menu/sidebar
   - If it's collapsed, click the **hamburger menu icon (≡)** at the top left
3. Find and click: **"APIs & Services"**
4. In the submenu, click: **"OAuth consent screen"**

## Method 2: Use the Search Box

1. At the **top of the page**, there's a search box that says "Search products and resources"
2. Type: **"OAuth consent screen"**
3. Click on "OAuth consent screen" in the search results

## Method 3: Direct URL

Copy and paste this URL directly into your browser:

```
https://console.cloud.google.com/apis/credentials/consent
```

## What You Should See

Once you're on the OAuth consent screen page, you'll see:
- A page titled "OAuth consent screen"
- Either:
  - A button that says "CREATE" or "CONFIGURE CONSENT SCREEN", OR
  - Configuration tabs if it's already been started

## If You See "OAuth consent screen is not configured"

- Click the button to configure it
- Follow the steps in the guide
