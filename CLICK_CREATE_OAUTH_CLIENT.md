# Click "Create OAuth client" Button

You found the "Create OAuth client" button! Here's what to do:

## Try Clicking It

Click the **"Create OAuth client"** button and see what happens.

### What Might Happen:

**Option A: Prompts to Configure Consent Screen**
- If OAuth consent screen isn't configured yet, Google will prompt you
- It will say something like "OAuth consent screen is not configured"
- Click the button/link to configure it
- Follow the configuration steps

**Option B: Goes Directly to Create Client**
- If consent screen is already configured, it goes to create credentials
- We can configure consent screen separately if needed

## Steps After Clicking:

1. **Click "Create OAuth client"**
2. **See what happens:**
   - Does it prompt you to configure consent screen?
   - Or does it go to a form to create the client?

## If It Prompts for Consent Screen:

Follow these steps:
1. Click the button to configure consent screen
2. Select "External" as user type
3. Fill in app information
4. Add scopes (gmail.readonly)
5. Add test users (amprenta.email@gmail.com)
6. Then come back to create the client

## If It Goes to Create Client Form:

We'll fill out:
- Application type: "Desktop app"
- Name: "Amprenta Gmail Client"
- Create and download
